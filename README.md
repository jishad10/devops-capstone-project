# Customer Accounts Microservice

[![Build Status](https://github.com/jishad10/devops-capstone-project/actions/workflows/ci-build.yaml/badge.svg)](https://github.com/jishad10/devops-capstone-project/actions/workflows/ci-build.yaml)

This is a RESTful microservice for managing customer accounts, built as part of the DevOps Capstone Project. It supports full CRUD operations (Create, Read, Update, Delete, List) and is developed using Test-Driven Development, deployed with CI/CD pipelines on Kubernetes.

## Overview

This project uses Flask, PostgreSQL, and Docker. It follows agile methodology with GitHub Kanban boards for sprint planning, and includes automated testing, linting (Flake8/Pylint), security headers, and CI/CD pipelines using GitHub Actions and Tekton.
